Surface texture using Ball-end Tool

What is it?

It is a texture method using micro-milling. Unlike conventional milling in which you may probably end up with simple grooves, the trochoidal movement of cutting edge is controlled by reference signals to create texture. To be specific, the spindle speed reference is modulated by giving harmonics signals to dSpace control box with MATLAB/Simulink.

Strategy

The overall strategy is shown in the figure below. Errors (run-out and deflections) are considered in the cutting edges trajectories.

Before doing any machining, I have written some codes to simulate the surface finish using Z-map method. The flow chart of the program in shown in the figure below.

The cutting force is simulated by converting cutting history points into a concave hull so that we can know the materials left (uncut chip thickness to be specific) on the workpiece for each time interval as shown in the figure below.

What have I learnt from this project?

Mostly electrical cabinet trouble shooting during the journey LOL. Giving x,y,z and spindle references may sound easy. But the CNC machine ran into problems serval times since it is built by previous students instead of commercial product as a whole.

Also implementing some simple logics in Simulink like safety shutdown, control mode switch etc. can be quite not straight-forward. But writting the surface finish simulation codes is the most enjoyable part of this project for sure. I have learnt some computer graphics concepets thought the process.
